Translator App



This is a full-stack web application built with Express, bcrypt, CORS, and other server-side technologies. The app serves as a Translator application, facilitating language translation from one language to another. Users can input text in a source language, select a target language, and then click the translate button to obtain the translated text. Under the hood, the app leverages the RapidAPI Text Translator API for language translation functionality, enhancing its capabilities and providing seamless translation experiences.



Pokemon Weather App



The Pokémon Weather App is a full-stack web application built with bcrypt, Express.js, CORS, and pg-promise. It combines the Pokémon API and the Weather API to offer users a unique experience. By inputting a location, users can access weather data specific to that location, which is then correlated with data from the Pokémon API to generate a list of Pokémon associated with the current weather conditions. Through this stack, the app ensures secure password handling, efficient database operations, and seamless integration of APIs for an immersive user experience.
